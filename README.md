# Baixando e configurando

- Abra um terminal:

    `Control"` + `Alt` + `t`

- Clone o repositório:

```bash
git clone https://github.com/LucasMateuss/frequencia-palavra.git
```

- Entre na pasta:
  
```bash
cd frequencia-palavra
```

- Instale as dependências:

```bash
poetry install
```

- Ative o ambiente virtual:

```bash
poetry shell
```

- Execute o script:

```bash
python frequencia.py
```

- Se preferir, abra o vscode:

```bash
code .
```
